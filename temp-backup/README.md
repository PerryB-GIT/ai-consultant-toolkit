# AI Consultant Toolkit

## Overview

An intelligent toolkit designed to compress client onboarding from 3-4 hours down to 60 minutes through AI-powered automation and structured workflows.

## Master Plan

Full project details and implementation strategy: [Master Plan](docs/plans/2026-02-13-ai-consultant-toolkit-master-plan.md)

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Language**: TypeScript
- **Deployment**: Vercel
- **AI Integration**: Claude Code, MCP Servers

## Setup Instructions

### Prerequisites
- Node.js 18+
- npm or pnpm
- Git

### Installation

```bash
# Clone the repository
git clone https://github.com/PerryB-GIT/ai-consultant-toolkit.git
cd ai-consultant-toolkit

# Install dependencies
npm install

# Run development server
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to see the application.
